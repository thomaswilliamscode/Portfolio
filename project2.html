Project Overview

I built this Todo App to sharpen my skills with React, TypeScript, and state management using the Context API. The goal was to go beyond a basic todo list by introducing nested folders and lists, URL-driven navigation, and persistent state with localStorage. Through this project, I reinforced concepts like component composition, dynamic routing with React Router, and type safety with discriminated unions. I also challenged myself to make the UI modular and easily extendable for future features like drag-and-drop reordering or Supabase sync. This project reflects my ability to design, build, and document a frontend feature set that balances clean code with practical functionality — the kind of work I’d bring to a production codebase.
